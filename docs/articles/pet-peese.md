# PET-PEESE model

## Introduction

PET-PEESE is a regression-based method for correcting small-study
effects.

## PET

Regression of test statistics on precision:

\[ t_i = + + u_i \]

## PEESE

Regression of effect size on squared standard error:

\[ y_i = + SE_i^2 + u_i \]

### Interpretation

The intercept estimates the effect as precision approaches infinity.
