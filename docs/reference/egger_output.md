# Assemble a `bayesma_egger` object from pipeline outputs

Assemble a `bayesma_egger` object from pipeline outputs

## Usage

``` r
egger_output(spec, fit, effects)
```

## Arguments

- spec:

  An `egger_spec` object.

- fit:

  An `egger_fit` object.

- effects:

  An `egger_effects` object.

## Value

A list of class `"bayesma_egger"`.
