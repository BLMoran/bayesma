# Assemble a `bayesma_metareg` object from pipeline outputs

Assemble a `bayesma_metareg` object from pipeline outputs

## Usage

``` r
meta_reg_output(spec, fit, effects)
```

## Arguments

- spec:

  A `meta_reg_spec` object.

- fit:

  A `meta_reg_fit` object.

- effects:

  A `meta_reg_effects` object.

## Value

A list of class `c("bayesma_metareg", "bayesma")`.
