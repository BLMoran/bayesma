# Update x-axis limits on the density plot

Update x-axis limits on the density plot

## Usage

``` r
sens_add_x_lim(x, xlim, x_breaks = NULL)
```

## Arguments

- x:

  A `bayesma_sensitivity_plot` object.

- xlim:

  Numeric vector of length 2 for x-axis limits.

- x_breaks:

  Optional numeric vector of break points.

## Value

Modified `bayesma_sensitivity_plot`.
