# Extract tidy effect components from a bivariate meta-analysis fit

Extract tidy effect components from a bivariate meta-analysis fit

## Usage

``` r
bayesma_mv_extract(fit, spec)
```

## Arguments

- fit:

  A `bayesma_mv_fit` object.

- spec:

  A `bayesma_mv_spec` object.

## Value

An object of class `"bayesma_mv_effects"`.
