# Assemble the final RoBMA output object

Assemble the final RoBMA output object

## Usage

``` r
robma_output(spec, fit, effects)
```

## Arguments

- spec:

  A `bayesma_robma_spec`.

- fit:

  A `bayesma_robma_fit`.

- effects:

  A `bayesma_robma_effects`.

## Value

An object of class `c("bayesma_robma", "bayesma")`.
