# Update titles on a sensitivity plot

Update titles on a sensitivity plot

## Usage

``` r
sens_add_titles(x, title = NULL, subtitle = NULL, align = NULL)
```

## Arguments

- x:

  A `bayesma_sensitivity_plot` object.

- title:

  Title string (or NULL to remove).

- subtitle:

  Subtitle string (or NULL to remove).

- align:

  Title alignment: "left", "center", or "right".

## Value

Modified `bayesma_sensitivity_plot`.
