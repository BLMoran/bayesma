# Assemble a `bayesma_mv` object from pipeline outputs

Assemble a `bayesma_mv` object from pipeline outputs

## Usage

``` r
bayesma_mv_output(spec, fit, effects)
```

## Arguments

- spec:

  A `bayesma_mv_spec` object.

- fit:

  A `bayesma_mv_fit` object.

- effects:

  A `bayesma_mv_effects` object.

## Value

A list of class `c("bayesma_mv", "bayesma")`.
