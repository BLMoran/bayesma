# Create INSPECT-SR Summary Bar Plot

Create INSPECT-SR Summary Bar Plot

## Usage

``` r
inspect_summary_plot(data, font = NULL, title = NULL)
```

## Arguments

- data:

  An `inspect_sr` object or data frame.

- font, title:

  As in
  [`inspect_plot`](https://blmoran.github.io/bayesma/reference/inspect_plot.md).

## Value

A ggplot2 object.
